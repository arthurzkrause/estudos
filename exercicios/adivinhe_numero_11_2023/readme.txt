Tarefa: Jogo de Adivinhação

Crie um jogo simples de adivinhação em Python. Aqui estão as instruções:

Escolha um número aleatório entre 1 e 20 (você pode usar a biblioteca random para isso).
Peça ao usuário para adivinhar o número.
Dê pistas se a tentativa foi muito alta ou muito baixa.
Continue pedindo ao usuário para adivinhar até que ele acerte o número.
Lembre-se de incluir mensagens amigáveis e informativas para guiar o usuário durante o jogo. 
Por exemplo, se o palpite do usuário for muito alto, você pode dizer "Tente um número mais baixo." 
Se o palpite for muito baixo, você pode dizer "Tente um número mais alto."


Este código implementa um jogo simples de adivinhação.

Gera um número aleatório entre 1 e 50, que é o número que o jogador deve adivinhar.
Inicia um loop onde o jogador é solicitado a fazer palpites para adivinhar o número secreto.
Verifica se o palpite do jogador é um número válido. Se não for, imprime uma mensagem informando que um número válido deve ser digitado.
Conta o número de tentativas.
Fornece dicas ao jogador, indicando se o palpite é muito alto ou muito baixo.
Quando o jogador adivinha corretamente, imprime uma mensagem parabenizando o jogador e informa o número de tentativas necessárias.
Se o jogador demorar 15 tentativas ou mais, imprime uma mensagem especial.
Limpa a tela entre cada tentativa para proporcionar uma melhor experiência visual.

O jogo continua até que o jogador adivinhe corretamente o número secreto ou decida sair do jogo.
