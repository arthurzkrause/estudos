Código:

Este código Python solicita ao usuário o número de leituras desejado. 
Em seguida, ele gera e imprime perguntas aleatórias do módulo texto_oracular até atingir o número de leituras especificado pelo usuário. 
O código usa um loop while com uma condição que verifica se a quantidade de leituras realizadas (quantidade) 
é diferente do número desejado de leituras (leituras). 
Cada pergunta é escolhida aleatoriamente usando a função random.choice e é exibida no console.

Astragaloi

O que eles são?
Basicamente, são dados antigos com apenas quatro lados!

Feito de ossos dos dedos de ovelha, vaca ou cabra ou réplicas de cerâmica deles
Eles foram lançados sobre uma mesa ou bandeja de dados. Essa bandeja era colocada 
sobre uma mesa em frente à imagem de um deus/deusa, geralmente Apolo ou Hermes.
Assim como os dados eram usados ​​para jogos, o mesmo acontecia com os astrágalos.

Como fazer uma leitura de astrágalo:
As perguntas feitas geralmente são pedidos de conselhos. Ambos os formatos de adivinhação são mais complexos do que sim/não.

Para este código uso a tabela de Jenna Mortensen "Astragaloi: Greco-Roman Dice Oracles"
