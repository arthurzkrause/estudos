Este código consiste em um sistema básico de criação de conta e login. 

Criação de Conta:
- Solicita ao usuário que insira um endereço de e-mail.
- Verifica se o e-mail contém o caractere '@'.
- Solicita ao usuário que insira uma senha e a confirme.
- Verifica se as senhas inseridas coincidem.
- Se todas as verificações são bem-sucedidas, a conta é considerada criada.

Login:
- Após a criação da conta, o usuário é apresentado com a opção de fazer login ou sair.
- Se o usuário escolher fazer login, é solicitado que insira seu endereço de e-mail e senha.
- Se as informações coincidirem com a conta criada anteriormente, uma mensagem de boas-vindas é exibida.
- Caso contrário, o usuário é informado de que as informações não conferem, e a opção de tentar novamente é fornecida.

O código utiliza uma abordagem simples, solicitando entradas do usuário, 
validando-as e fornecendo feedback adequado. 
A função limpar_tela() é usada para limpar a tela do console, proporcionando uma experiência mais organizada ao usuário.

Observação:
É importante observar que este código é uma implementação básica e educacional, 
adequada para entender os conceitos fundamentais de entrada do usuário, 
validação e estruturas de controle em Python. Em um ambiente de produção, 
medidas adicionais, como o armazenamento seguro de senhas, seriam necessárias para garantir a segurança das informações da conta.
