Esse código em  implementa um programa simples de lista de tarefas interativo.

Inicia uma lista vazia chamada lista_completa.
Entra em um loop infinito que exibe um menu de opções para o usuário.
O usuário pode escolher entre quatro opções:
    [i]nserir: Permite ao usuário adicionar um item à lista.
    [a]pagar: Permite ao usuário apagar um item da lista, se existir na lista.
    [l]istar: Lista todos os itens da lista numerados.
    [c]oncluir: Encerra o programa.
O programa executa a ação correspondente à escolha do usuário.
    Se o usuário escolher listar a lista, verifica se a lista está vazia antes de exibi-la.
    Se o usuário escolher concluir, o programa sai do loop, encerrando a execução.
O código utiliza a biblioteca "os" para limpar a tela do console (os.system('cls')) 
e fornece feedback ao usuário sobre as ações realizadas na lista.
